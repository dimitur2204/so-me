SEO Guide

On Page SEO

  • Title element: example: Guide for International Students in Denmark

    Use the main keywords that would capture the whole idea of the future product (guide, international, students)

  • Meta description

    A sentence or two including as much of the keywords as possible Example: Thinking of studying in Denmark in a University or Academy. We explain all about moving in, find housing, degrees and top ups, tips and tricks, SU, career options, documents such as CPR and all you need for the SU.

  • Short and descriptive URL (hard to do since we use github pages)

    We can imagine the clients don’t use github pages and we can think of a normal domain and url

  • Responsive/mobile friendly

    We should use a CSS library to make it easy to make the site responsive

  • Internal linking

    Make anchors to subsections in the website (for example - from how to get a CPR number → how to get SU)
